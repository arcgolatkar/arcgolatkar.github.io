# Archit Golatkar - Portfolio

Minimal portfolio site for GitHub Pages.

## Test Locally

```bash
# Option 1: Python
python3 -m http.server 8000

# Option 2: Node.js (if you have npx)
npx serve .
```

Then open http://localhost:8000 in your browser.

## Deploy

Push to the `main` branch. GitHub Pages serves from the root automatically for `username.github.io` repos.
